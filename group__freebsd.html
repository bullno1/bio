Detailed Description

FreeBSD implementation details.

bio uses uses kqueue on FreeBSD.

Abstract socket is not supported. Using it will result in ENOTSUP.

For file, aio is used. However, it is possible that unsafe operations on special files might not be enabled with vfs.aio.enable_unsafe=1. For each file, bio will detect EOPNOTSUPP on the initial aio call and fallback to the async thread pool.
