Detailed Description

A logger that writes messages to a file.

bio_logger_t logger = bio_add_file_logger(&(bio_file_logger_options_t){
.file = BIO_STDERR,
.with_colors = true,
});
BIO_INFO("Started");
// Main loop
while (running) {
// ...
}
// Flush messages and terminate the logger coroutine
bio_remove_logger(logger);

Remarks
This will spawn a coroutine to handle writing and avoid making the calling coroutine context switch due to I/O. The returned bio_logger_t should be removed with bio_remove_logger at the end of a program to ensure that bio_loop will terminate.
